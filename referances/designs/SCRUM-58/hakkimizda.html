Implementation Notes for Developers

OPTIONAL

1. Team/Leadership Section

  • New component: TeamSection.tsx — inserted between MediazoneStats and MediazoneFamily
  • Grid: grid-cols-2 md:grid-cols-3 lg:grid-cols-6
  • Background: bg-primary-50
  • Avatar: 80px circle with gradient, white initials. Each member gets a unique gradient color pair.
  • Card hover: gold border #D4AF37 + subtle lift translateY(-2px)
  • Requires Human PM approval before implementation. Team member names/roles are placeholders.
UPGRADE

2. MediazoneFamily Card Upgrade

  • Monogram container: w-14 h-14 rounded-xl bg-{color}-100 → w-16 h-16 rounded-full bg-gradient-to-br from-{color}-500 to-{color}-600
  • Text color: colored initials on light bg → white text on gradient bg
  • Brand name: font-medium text-primary-600 → font-semibold text-primary-900
  • Added category label: text-[11px] text-primary-400 (e.g., "Dijital Medya", "Haber", "Spor")
  • Hover: gold border + lift animation (same as team cards)
  • Keep 6-column grid layout on desktop
BUG FIX

3. "İletişime Geç" Button Fix

  • Location: AboutCTA section, secondary button
  • Change: href="#" → href="/iletisim"
  • Element: The <a> tag wrapping "İletişime Geç" text
  • This was flagged by tester as only CTA text mismatch on the about page
ANIMATION

4. Scroll-Reveal Animations

  • Apply to: All sections below hero (MediazoneStats, Team, MediazoneFamily, OfficeLocations, AboutCTA)
  • Animation: fade-in-up — opacity 0→1, translateY(24px→0), duration 0.6s, ease-out
  • Trigger: IntersectionObserver with threshold 0.15
  • Staggered children: grid items get +0.1s delay per item (e.g., stat cards, team cards, family cards)
  • Elements start with opacity: 0; transform: translateY(24px), class .revealed applies the animation
  • Run once per element — remove observer after animation triggers
