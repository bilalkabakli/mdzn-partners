SCRUM-58 Change Summary — Influencer'lar İçin

1. Hero Right Panel — 3 Vertical Stat Rows (CRITICAL FIX)

The implementation currently uses a 2x2 grid (grid-cols-2) identical to Brands and Ajanslar pages. The design specifies 3 stacked rows with space-y-4:

  • Each row: flex justify-between items-center
  • Row 1: Bu Ay Kazanç: ₺8,450 — TrendingUp icon, success-600 green icon bg
  • Row 2: Toplam Sipariş: 127 — ShoppingBag icon, neutral bg
  • Row 3: Aktif Kampanya: 8 — Megaphone icon, accent-600 gold icon bg
  • Container: glass card (bg rgba(255,255,255,0.1), backdrop-blur 12px, border rgba(255,255,255,0.15))

2. Hero CTA — Accent Button (VERIFIED)

CTA must be accent-colored (gold #D4AF37), NOT white. Text: "Aramıza Katıl" with arrow icon.

3. Decorative Blur Circles (VERIFIED)

Background includes 2 decorative blur circles (accent-600/10 + primary-700/30) for visual depth.

4. "Hemen Başvur" CTA in Bottom Section

Annotated to open the application modal (SCRUM-52 component).

5. Hero Padding (ALIGNED)

Using py-12 lg:py-16 per design system spec.
